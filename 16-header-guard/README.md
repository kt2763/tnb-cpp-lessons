# C++講義 #16 pragma once とヘッダガード

📺 **動画**: [C++講義#16 pragma onceとヘッダガード](https://youtu.be/arKM6wudtcE)

## 内容

ヘッダファイルの多重インクルードを防ぐ `#pragma once` と、
`#ifndef` / `#define` / `#endif` によるヘッダガードの書き方を学びます。

- `header-guard-0/` — `Player.hpp` / `Enemy.hpp` / `main.cpp` を使った多重インクルードの例
- `header-guard-1/` — ヘッダガードを適用した `Player.hpp` の例
